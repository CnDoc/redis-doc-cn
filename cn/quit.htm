QUIT

Related commands

  • AUTH
  • ECHO
  • PING
  • QUIT
  • SELECT

Available since 1.0.0.

Ask the server to close the connection. The connection is closed as soon as all pending replies have been written to the client.

Return value

Simple string reply: always OK.
